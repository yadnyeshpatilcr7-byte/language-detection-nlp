# Language Detection using NLP

## 📌 Overview
This project detects the language of input text using Natural Language Processing and Machine Learning.

## 🚀 Features
- Text preprocessing
- TF-IDF vectorization
- Naive Bayes & Logistic Regression models
- Model comparison
- Custom language prediction

## 📊 Results
- Naive Bayes Accuracy: ~68%
- Logistic Regression Accuracy: ~73%

## 🛠 Technologies Used
- Python
- Scikit-learn
- NLTK
- Pandas
- Matplotlib

## 📁 Dataset
Kaggle Language Detection Dataset

## ▶️ How to Run
1. Install dependencies
2. Run notebook in Google Colab / Jupyter

## 📌 Conclusion
Logistic Regression performed better than Naive Bayes for this task.
